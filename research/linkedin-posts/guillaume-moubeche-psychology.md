# Guillaume Moubeche: Copy/Outreach Frameworks (LinkedIn)

## Framework 1: “Permission Ask”
- **Post Template**
  - 1 line context (“Noticed X about your team/process…”)
  - 1 line insight (“Usually that means Y risk/opportunity…”)
  - Soft ask (“Worth me sending a 5-line breakdown?”)
- **The Psychology**
  - A permission ask lowers threat and shifts the brain from “defend” to “evaluate.”
- **Result**
  - More replies from skeptical buyers because the recipient can say “yes/no” without committing to a meeting.

## Framework 2: “Micro-Audit (value first)”
- **Post Template**
  - “I looked at X and found 3 things…”
  - Bullet 3 findings (specific + short)
  - Close with a question (“Want the fix list?”)
- **The Psychology**
  - Reciprocity + specificity: if you did real work, you’re not “just another sender.”
- **Result**
  - Higher conversion from reply → call because the thread already contains evidence you can help.

## Framework 3: “Anti-Pitch Story”
- **Post Template**
  - Story: “We stopped doing X. Here’s what happened.”
  - 2–3 lessons
  - Invite discussion (“Curious how you’re handling this?”)
- **The Psychology**
  - Stories reduce sales resistance; readers learn without feeling sold to.
- **Result**
  - Better inbound + warmer outbound because you’re remembered as an operator, not a spammer.

## 100Hires Application
100Hires can apply these frameworks to reduce friction in outbound:
- Use **permission asks** to start conversations with HR leaders without forcing a demo.
- Send **micro-audits** (e.g., “3 hiring workflow bottlenecks we see in X industry”) to create proof-based threads.
- Publish “anti-pitch” operator stories (what you stopped doing in recruiting ops) to build authority and make cold outreach feel familiar.
